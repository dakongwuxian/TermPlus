# 本示例使用while循环进行了发送和在主窗口中插入字符串的操作
# 脚本需要遵守python的缩进规则，python通过缩进来识别哪些属于一个大括号（if/for/while等）中的内容。
# 缩进建议全部使用4n个空格或者全部使用tab。

i=0
while i<3:
    #send command1 # 需要执行的操作，在本示例中被注释掉
    show_str("external loop count is %d\n"%i) # 在主窗口显示当前循环次数
    wait(1)
    i=i+1
    j=0
    while j<3:
        #send command2
        show_str("internal loop count is %d\n"%j)
        wait(1)
        j=j+1


# 可通过上面的形式，将循环相关的变量也包含在发送的命令中
