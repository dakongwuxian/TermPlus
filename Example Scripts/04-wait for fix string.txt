# 本示例使用wait for函数对主窗口中出现的内容进行了等待和识别，再加上if语句，实现了当出现特定字符串时的指定操作。
# 脚本需要遵守python的缩进规则，python通过缩进来识别哪些属于一个大括号（if/for/while等）中的内容。
# 缩进建议全部使用4n个空格或者全部使用tab
# wait for xxx for t 函数会在运行后开始识别主窗口中新出现的内容，如果发现了指定的字符串xxx，则会停止运行，开始执行后面的函数，如果一直没找到指定字符串，则会在t秒之后结束等待。
# wait for函数是有返回值的，如果发现了指定字符串，返回值为1(True)，没发现返回值为0(False)。可以利用返回值配合if进行指定操作。
	
# 发送需要判断回复内容的命令

send lmclist

wait_result = wait_for('CXP2021113/1_R24B49',3)

if wait_result==True:

    # 在主窗口插入字符串，以便在log中查找
    show_str("Target found.\n")

    # pop out message # 弹出窗口提示
    messagebox.showinfo("提示", "Target string found.")
    
    # 找到了期待的回复，打印log记录下来，此示例中未运行
    #send te log read
    
    # 停止脚本运行
    raise StopLoopException("stop")
else:
    show_str("Search over time, target not found.\n")
    messagebox.showinfo("提示", "Target string not found.")




# 示例二
# 本示例使用get_future_lines_till_str_or_overtime(）获取了主窗口中的内容，使用re.search(）实现了对指定字符串的搜索，使用if对搜索结果进行了判定，当主窗口出现指定内容时，执行指定的操作。
# 脚本需要遵守python的缩进规则，python通过缩进来识别哪些属于一个大括号（if/for/while等）中的内容。
# 缩进建议全部使用4n个空格或者全部使用tab。

# 发送命令
send lmclist

# 获取文本区域从现在开始新出现的所有内容,直到出现字符串krypton:~#，或者20秒之后还没出现也停止
response = get_future_lines_till_str_or_overtime("krypton:~#",20) 

# 设置想要查找的文本
match_target = "CXP2021113/1_R24B49"

# 根据查找的结果，进行后续操作
if match_target in response: 
    messagebox.showinfo("提示", "Target string found.")# pop out message
    raise StopLoopException("stop")# stop running
else:
    messagebox.showinfo("提示", "Target string not found.")# pop out message
