# 本示例使用for循环进行了发送和在主窗口中插入字符串的操作
# 脚本需要遵守python的缩进规则，python通过缩进来识别哪些属于一个大括号（if/for/while等）中的内容。
# 缩进建议需要全部使用4n个空格或者全部使用tab

for i in range(10):
    #send command1 # 需要执行的操作，在本示例中被注释掉
    show_str("external loop count is %d\n"%i) # 在主窗口显示当前循环次数
    wait 1    
    for j in range(10):
        #send command2 # 需要执行的操作，在本示例中被注释掉
        show_str("internal loop count is %d\n"%j) # 在主窗口显示当前循环次数
        wait 0.3

# 如果send后面发送的内容包含单引号或双引号，需要如下处理

# 内含单引号 (It's)	"It's a test"
# 内含双引号 ("Hello")	'He said "Hello"'
# 同时含单/双引号	'''He said, "It's fine."'''
# 风格统一但要转义	"She said, \"Hello\" and I replied \"Hi\""


# 下面的2个例子展示了如何从 rvc r PaDriverBias0:0 发送到 rvc r PaDriverBias0:63

for i in range(64):
    send_line("rvc r PaDriverBias0:%d"%i)
    wait 1

for i in range(64):
    send_line(f"rvc r PaDriverBias0:{i}")
    wait(1)

# 下面的2个例子展示了如何发送变化的数字在中间的情况
